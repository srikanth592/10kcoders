Selectors

  • Selector:means element selector which use select our element wants styling. i.e h,p, and other tags.

  • Property:property which use to style the element i.e color,backgroundcolor,textassign

  • Value:assigning a value for select property i.e color red other.

  • Inline css:which is used to write styling a tag in itself.

  • External css:which is used call a file by link tag in head which wrote a css attach file with styling.

  • Internal css:mean writing a code inside head tag for styling a webpage.

  • Element selector: mean only selecting a single element which selcted tag wil be style.

  • Group selector:instead of selecting single element it call multiple element at one time which to be style.

  • Universal selector: selecting whole web page by * before the bracess.

  • Note:priority wise inline css->internal css->external css.
