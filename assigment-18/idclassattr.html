Difference between Id and Class and Attribute Selectors



Id Class Attribute
Syntax:In html for an element the id name starts with the '#' symbol followed by a unique name assigned to it. Syntax:Class assigned to an element has its name starts with '.' followed by class name. The [attribute] selector is used to select element with a specified attribute.
Selector:only one id selector can be attached to a element. Multiple class selectors can be attached to an element. input[attributename] - it will check if that attribute is present ,only then it will apply style.
Id unique in a page and can only apply to at most one element. The class can be applied to multiple element so it could be multiple times on a single page.
  • input[attribute="value"] - it will check which element has that attribute value and applies style.

  • input[attribute~="word"] - it will search for that word in attribute value,it will apply style.

  • input[attribute^="word"] - checks attribute values which start with that word,it will apply style.

  • input[attribute$="word"] - checks attribute value which ends with that word and applies style.
