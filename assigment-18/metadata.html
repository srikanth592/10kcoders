METADATA tags:

  • Metadata:meta mean metadata is data about html document.

  • Charset:specifies the charcter encoding for the html document.

  • X-UA-Compatible:is compatible with all over versions of internet.

  • IE=edge:is used the latest edge versions of rendering engine.

  • Viewport:is the user's interface od web page.

  • width=device-width:part set the width of the page to follow the screen width of the device.

  • initial-scale=1.0:sets the initial zoom level when the page is first loaded by the browser.
