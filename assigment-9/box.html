Selector

Selector means element selector which use select our element wants styling. i.e h,p, and other tags.

Property

property which use to style the element i.e color,backgroundcolor,textassign.

Value

assigning a value for select property i.e color red other.

Inline css

inline is used to write styling a tag in itself.

External css

external is used call a file by link tag in head which wrote a css attach file with styling.

Internal css

Internal mean writing a code inside head tag for styling a webpage.

Element selector

mean only selecting a single element which selcted tag wil be style.

Group selector

Instead of selecting single element it call multiple element at one time which to be style.

Universal selector

selecting whole web page by * before the bracess.


Note:priority wise inline css->internal css->external css.

ID Selector

The id selector uses the id attribute of an HTML element to select a specific element.The id of an element is unique with in a page,so the id selector is used to select one unique element.To select an element with a specific id ,write a hash(#) character,followed by the id of the element.

Class Selector

The class selector select HTML element with a specific class attribute.To select element with a specific class,write a period (.) character,followed by the class name

Attribute Selector

The attribute selector is used to select element with a specified attribute and value.

[Attribute] Selector

Input[attributename] - it will check if that attribute is present ,only then it will apply style.

[Attributename="value"] Selector

Input[attribute="value"] - it will check which element has that attribute value and applies style.

[Attributename~="value"] Selector

Input[attribute~="word"] - it will search for that word in attribute value,it will apply style.

[Attributename^="value"] Selector

Input[attribute^="word"] - checks attribute values which start with that word,it will apply style.

[Attributename$="value"] Selector

Input[attribute$="word"] - checks attribute value which ends with that word and applies style.

[Attributename*="value"] Selector

the selector is used to select elements whose attribute value contains a specific value.

Combinator Selector

A CSS selector can contain more than one simple selector. Between the simple selectors, we can include a combinator.

Descendant Selector

The descendant selector matches all elements that are descendants of a specified element. ex:div p{ }

Child Selector

The child selector selects all elements that are the children of a specified element. ex div > p{ }

Adjacent Sibling Selector

The adjacent sibling selector is used to select an element that is directly after another specific element. Sibling elements must have the same parent element, and "adjacent" means "immediately following". Ex:div + p{ }

General Sibling Selector

The general sibling selector selects all elements that are next siblings of a specified element. ex:div ~ p{ }
